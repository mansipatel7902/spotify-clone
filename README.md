# 🎵 Spotify Clone

A responsive Spotify Clone built using **HTML** and **CSS**, inspired by Spotify's web player interface.

## 🚀 Features

- Responsive design
- Modern UI
- Sidebar navigation
- Music cards with hover effects
- Clean layout

## 🛠️ Technologies Used

- HTML5
- CSS3

## ▶️ How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
